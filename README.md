# 📊 Customer Churn Analysis

## 🔍 Overview
This project analyzes customer churn behavior using both Excel and Python to identify key factors affecting customer retention. The analysis is performed on a telecom dataset and combines Excel dashboards with Python-based data analysis and visualization.

---

## 🎯 Objectives
- Calculate overall churn rate  
- Analyze churn based on contract type  
- Study churn trends across customer tenure  
- Build visual dashboards for better insights  

---

## 🛠️ Tools & Technologies
- **Excel**: Pivot Tables, Dashboard  
- **Python**: Pandas, Matplotlib  
- **Google Colab**

---

## 📊 Key Insights
- **Overall churn rate is ~27%**, indicating significant customer loss  
- **Month-to-month contracts** have the highest churn  
- **Early tenure customers (0–12 months)** are most likely to churn  
- Long-term contracts improve customer retention  

---

## 📁 Project Files
- `Customer_Churn_Analysis.py` → Python analysis  
- `Customer_Churn_Dataset.xlsx` → Dataset  
- Charts → Visualizations generated from Python  

---

## 📈 Visualizations
- Pie Chart → Overall Churn  
- Bar Chart → Churn by Contract Type  
- Bar Chart → Churn by Tenure  

---

## 🚀 How to Run
1. Open the Python file in Google Colab  
2. Upload the dataset (Excel file)  
3. Run the code  
4. Charts and insights will be generated  

---

## 💡 Conclusion
The analysis highlights that contract type and early customer engagement are critical factors influencing churn. Businesses can reduce churn by encouraging long-term contracts and improving onboarding experience.
